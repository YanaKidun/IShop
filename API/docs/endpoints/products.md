# GET /products

## Описание метода
Получение списка всех товаров в магазине.

## Тело запроса
Нет тела запроса.

## Тело ответа
```json
[
  {
    "id": 105,
    "name": "Молоко",
    "price_with_discount": 1.99,
    "group": "Молочные продукты",
    "char_code": "SHP0001",
    "count_of_warehouse": 1000
  }
]
 ```

## Описание ошибок:
500: Внутренняя ошибка сервера

# POST /products

## Описание метода
Создание нового товара. Доступно только для экономиста.

## Тело запроса
```json
{
  "name": "Молоко",
  "price_with_discount": 1.99,
  "group": "Молочные продукты",
  "char_code": "SHP0001",
  "count_of_warehouse": 1000
}
 ```

## Тело ответа
нет

## Описание ошибок:
400: Некорректные данные
500: Внутренняя ошибка сервера


# GET /products/{id}

## Описание метода
Получение информации о конкретном товаре по его ID.

## Тело запроса
Нет тела запроса.

## Тело ответа
```json
{
  "id": 105,
  "name": "Молоко",
  "price_with_discount": 1.99,
  "group": "Молочные продукты",
  "char_code": "SHP0001",
  "count_of_warehouse": 1000
}
 ```

## Описание ошибок:
404: Товар не найден
500: Внутренняя ошибка сервера
